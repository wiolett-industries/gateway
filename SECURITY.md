# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Gateway, please report it responsibly.

**Do not open a public issue.**

Instead, email [contact@wiolett.net](mailto:contact@wiolett.net) with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect an initial response within 72 hours.

## Supported Versions

Security updates are provided for the latest release only.

## Scope

This policy covers the Gateway application code. Third-party dependencies are tracked separately — if you find a vulnerability in a dependency, please report it upstream and notify us so we can update.
